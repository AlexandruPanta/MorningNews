# MorningNews

## Overview
MorningNews is a project. This repository is dedicated to providing users with the latest news updates every morning.

## Features
- Daily news updates
- User-friendly interface
- Easy navigation

### Installation

```bash
yarn install // backend / frontend
yarn start // backend
yarn dev // frontend
```

## Tech Stack

* MongoDB
* Express
* Node.js
* React.js
* Next.js
